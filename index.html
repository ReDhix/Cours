La page web est obtenue par interprétation d'un code HTML envoyé par le serveur, réponse à la requête qui lui est envoyée (adresse en http://www.etc). C'est le navigateur qui interprète le code.

Il existe deux sortes de balises :

  1. Les balises les plus courantes encadrent du code qui se termine par une balise dite fermante. Exemple : </balise>contenu HTML
    2. </balise>
  2. Les balises dites orphelines qui n'encadrent aucun code et qui se ferment toutes seules. Exemple : </br> ; </hr> ; </img> ;</input> etc...
